Speech-to-Text to Braille Printing System

An assistive technology project designed to help visually impaired individuals by converting spoken language into Braille text automatically. This system uses offline speech recognition to capture user speech, transforms it into editable text, converts the text into Braille characters, and sends the output to a Braille printer through Bluetooth communication.

The project is built using an Android application with a user-friendly interface and supports continuous real-time voice transcription without requiring an internet connection. An Arduino-based hardware module receives the converted Braille data and controls the Braille printing mechanism.

Key Features
Offline speech-to-text conversion
Real-time continuous voice recognition
Text-to-Braille translation
Bluetooth connectivity with Braille printer
Android mobile application interface
Fast, portable, and user-friendly system
Technologies Used
Android Studio
Kotlin / Jetpack Compose
Vosk Speech Recognition
Arduino Uno
HC-05 Bluetooth Module
Braille Printing Mechanism
Objective

To provide an affordable, portable, and efficient communication tool that enables visually impaired users to access spoken information in Braille format, improving education, accessibility, and independence.
